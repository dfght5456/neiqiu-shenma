# 内丘神码数字化展示平台视觉优化需求文档

> 版本：v1.0  
> 日期：2026-05-31  
> 项目：内丘神码数字化展示平台  
> 目标：全面提升界面高级感，优化移动端用户体验

---

## 一、项目背景与现状分析

### 1.1 项目定位

内丘神码数字化展示平台是一个展示河北省非物质文化遗产——"内丘神码"（传统民间宗教祭祀木刻版画）的文化传播平台。项目面向大众普及非遗知识，主要受众为对传统文化感兴趣的年轻群体。

### 1.2 现存问题分析

| 问题类别 | 具体表现 | 影响程度 |
|---------|---------|---------|
| **视觉风格** | 界面风格偏普通，缺乏文化特色与高级感 | 高 |
| **色彩体系** | 色彩运用较为基础，未形成统一的品牌色系 | 高 |
| **排版层次** | 信息层级不够清晰，主次关系不明显 | 中 |
| **交互反馈** | 微交互细节不足，缺乏细腻的动效反馈 | 中 |
| **移动端适配** | 部分页面在移动端显示效果不够精致 | 高 |
| **文化氛围** | 传统元素融入不够深入，文化氛围感不足 | 高 |

## 三、视觉优化总体策略

### 3.1 设计理念

**"守正创新，神韵天成"**

- **守正**：守住内丘神码的文化内核，将神码的纹样、色彩、造型融入界面
- **创新**：运用现代UI设计手法，打造高级感与易用性兼具的界面
- **神韵**：传递神码艺术的古朴韵味与民间审美

### 3.2 核心设计原则

1. **文化元素具象化**：将神码纹样（祥云纹、回纹、水波纹等）转化为UI组件
2. **色彩体系品牌化**：建立以神码传统色为核心的专属配色方案
3. **层次结构清晰化**：通过字号、粗细、深浅区分信息主次
4. **交互体验细腻化**：增加微动效、状态反馈、手势引导
5. **移动体验优先化**：针对移动端进行专项优化

### 5.2 间距系统

采用 **8px 基础单位** 间距系统：

```css
/* 间距变量 */
--spacing-xs: 4px;    /* 紧密间距 - 同行元素间 */
--spacing-sm: 8px;    /* 小间距 - 相关元素间 */
--spacing-md: 16px;   /* 中间距 - 内容与边框间 */
--spacing-lg: 24px;   /* 大间距 - 模块内区块间 */
--spacing-xl: 32px;   /* 超大间距 - 模块与模块间 */
--spacing-2xl: 48px;  /* 页面级间距 - 主要区块间 */

/* 模块间距规则 */
模块间距 = 内容间距 × 2
/* 例如：卡片内padding为16px，卡片间距应为32px */
```

### 5.3 对齐规则

| 元素类型 | 对齐方式 | 说明 |
|---------|---------|------|
| 文字内容 | 左对齐 | 符合阅读习惯 |
| 标题文字 | 左对齐或居中 | 根据页面类型选择 |
| 图片内容 | 居中对齐 | 单图展示 |
| 卡片列表 | 左对齐 | 多卡片排列 |
| 操作按钮 | 右对齐或居中 | 主要操作居中，次要操作右侧 |
| 图标组 | 居中对齐 | 功能入口图标 |

### 8.1 页面过渡动效

```css
/* 页面进入动画 */
.page-enter {
  animation: pageEnter 0.4s ease-out;
}

@keyframes pageEnter {
  from {
    opacity: 0;
    transform: translateY(20px);
  }
  to {
    opacity: 1;
    transform: translateY(0);
  }
}

/* 页面退出动画 */
.page-exit {
  animation: pageExit 0.3s ease-in;
}

@keyframes pageExit {
  from {
    opacity: 1;
    transform: translateY(0);
  }
  to {
    opacity: 0;
    transform: translateY(-20px);
  }
}
```

### 8.2 卡片交互动效

```css
/* 卡片hover效果 */
.card-interactive {
  transition: all 0.3s ease;
}

.card-interactive:hover {
  transform: translateY(-4px);
  box-shadow: 
    0 8px 24px rgba(0, 0, 0, 0.08),
    0 12px 32px rgba(0, 0, 0, 0.04);
}

/* 卡片点击效果 */
.card-interactive:active {
  transform: translateY(-2px);
  transition-duration: 0.1s;
}
```

### 8.3 滚动触发动效

```javascript
// 使用AOS库实现滚动动画
AOS.init({
  offset: 100,        // 触发距离
  duration: 600,      // 动画时长
  easing: 'ease-out', // 缓动函数
  once: true,         // 只触发一次
});

// 动画类型配置
// data-aos="fade-up"      - 上浮淡入

// data-aos-delay="100"    - 延迟触发
```
